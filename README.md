# Knowledge-Vault

A clean and simple HTML & CSS website for organizing personal knowledge, notes, and learning resources.

---

## Features
- Structured thinking, not just note-taking
- Keep notes, links & learning summaries in one place
- Connect related ideas automatically
- Build lasting knowledge with reflection prompts
- Fast search to find anything instantly
- Clean UI with clear call-to-action

---

## Technologies
- HTML5  
- CSS3

---

## Folder Structure



Knowledge-Vault/

## Purpose
This project was created as part of an HTML & CSS assignment to practice semantic HTML, CSS layout, and responsive web design. It focuses on building a clean, well-structured landing page with a functional layout.
